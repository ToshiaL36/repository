Styling Links

With CSS, links can be styled in different ways. Links can be styled with any CSS property (e.g. color, font, background,etc).

In addition, links can be styled differently depending on what state they are in. the four links States are:

    a: link- a normal, unvisited link
    a link the user has visited
    a: hover- a link when the user mouses over it
    a: active- a link the moment it is clicked

When setting the style for several link states, there are some order rules:

    a: hover MUST come after a:link and a:visited
    a: active must come after a:hover
