Css Outline

The CSS outline properties specify the style, color and width of an outline.

An out line is a line that is drawn around elements(outside the borders) to make the element "stand out".

However, the outline property is different from the border property- The outline is NOT a part of an element's dimensions; the element's total width and height is not affected by the with of the outline.
